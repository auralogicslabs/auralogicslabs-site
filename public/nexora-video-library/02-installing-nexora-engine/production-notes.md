# Video 02: Installing Nexora Engine

## Target Length

3-5 minutes.

## Recording Flow

1. WordPress Admin > Plugins > Upload Plugin.
2. Select the Nexora Engine zip.
3. Install and activate.
4. Walk through the setup wizard:
   - Server type detection: Nginx / Apache.
   - WP_CACHE confirmation.
   - advanced-cache.php created automatically.
   - First snapshot capture starts.
5. Open the dashboard:
   - Cache Hit Ratio.
   - TTFB card.
6. Visit the public site in incognito.
7. Show response headers and the 22ms static delivery result.

## Voiceover Close

Your site is now serving static HTML. That's it.
