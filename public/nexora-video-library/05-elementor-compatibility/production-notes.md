# Video 05: Elementor Compatibility

## Target Length

90 seconds.

## Tone

Calm, reassuring, peer-to-peer.

## Script Beats

1. Will my sliders break?
   - No. Nexora captures the fully rendered DOM including Elementor CSS and JS.
2. Will my forms stop working?
   - Forms can fall back to live PHP rendering.
3. Can I still edit in Elementor?
   - Logged-in users always see live PHP and normal Elementor editing.
4. What if I need to revert?
   - One toggle. Instant revert. No content changes.

## Ending

Show the Static Delivery toggle in Nexora admin and a live Elementor page example.
