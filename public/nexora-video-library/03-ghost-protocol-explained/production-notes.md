# Video 03: Ghost Protocol Explained

## Target Length

2 minutes.

## Recording Flow

1. Show simulated before cURL output:
   - WordPress version hints.
   - REST endpoint.
   - X-Powered-By.
   - wp-content paths.
2. Enable Ghost Protocol:
   - Nexora Admin > Security.
   - Toggle Ghost Protocol on.
3. Run the same cURL command:
   - Headers are stripped or replaced.
   - WordPress fingerprints are reduced.
4. Explain:
   - `window.wp` becomes `window.ncx`.
   - `/wp-content/` paths are masked.
   - WordPress fingerprints are removed from public responses.
